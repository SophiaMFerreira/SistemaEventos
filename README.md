<h1 align="center">Sistema de Eventos 🥂</h1>
    Versão: 1.0 <br>
    Data: 12/09/2025

## Elaborador por: Sophia Agnes e Thaís Sampaio

O sistema de eventos é uma plataforma web destinada ao gerenciamento completo do ciclo de vida de eventos, abrangendo desde o cadastro de participantes, organizadores e administradores gerais até a realização e acompanhamento das atividades relacionadas. A solução possibilita que organizadores, sejam pessoas físicas ou organizações, cadastrem e administrem eventos em diferentes modalidades (presencial, online ou híbrida), com controle de inscrições, emissão de ingressos, acompanhamento da lotação, cancelamentos e geração de relatórios. O sistema oferece mecanismos de busca avançada, dashboards administrativos, exportação de relatórios em diferentes formatos e notificações automáticas, garantindo segurança, transparência e conformidade com a legislação vigente, incluindo a proteção de dados pessoais.

### Escopo

O sistema permite que:

- Organizadores crie e gerencie eventos;

- Participantes se inscrevam e acompanhem os eventos;

- O sistema controla os ingressos e pagamentos.

## ⚙️ Pré-requisitos

Antes de iniciar a instalação, verifique se você possui os seguintes itens instalados:

- Node.js

- NPM

- Git

## 🚀 Instalação

1- Clone o repositório:
- `https://github.com/SophiaMFerreira/SistemaEventos.git`

2 - Entre na pasta do projeto
- `cd SistemaEventos`

3- Instale as dependências
- `npm install`

4- Inicie a aplicação
- `npm start`
